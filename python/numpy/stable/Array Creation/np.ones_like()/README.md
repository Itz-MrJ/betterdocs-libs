# NumPy Array Creation Methods

## `np.ones_like`

The `np.ones_like` method creates a new array with the same shape and type as a given prototype array, but with all elements set to one.

### Example

```python
import numpy as np

arr = np.ones_like(np.array([1, 2, 3]), dtype=int)
```

For a `detailed` usage, check out [BetterDocs - np.ones_like()](https://betterdocs.tech/python/libs/numpy/stable/creation/ones_like)
